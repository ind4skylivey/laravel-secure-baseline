# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-03

### Added
- Initial public release of Laravel Secure Baseline.
- Artisan commands `secure:scan` and `secure:report` with JSON, Markdown, HTML, and SARIF outputs.
- Core scanners for configuration, sessions/cookies, CORS, security headers, debug routes (Telescope/Horizon/phpinfo), and dependency versions.
- CI/CD examples for GitHub Actions, GitLab CI, and generic shell usage.
- Documentation and badges for PHP 8.2+, Laravel 10/11, license, tests, Packagist version, and downloads.
